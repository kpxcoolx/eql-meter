# Security Policy

## Supported versions

Security fixes go to the latest release on the `main` branch.

## Reporting a vulnerability

Please **do not** open a public issue for security problems.

Email or message the maintainer privately, or use [GitHub Security Advisories](https://github.com/kpxcoolx/eql-meter/security/advisories/new) if available.

Include:

- What the issue is
- Steps to reproduce
- Impact (e.g. local file access, unexpected network use)

## Scope notes

EQL Meter is a local desktop app. It reads EverQuest Legends log files on your machine and does not require an account or cloud backend. Treat any unexpected network activity as suspicious and report it.
